##📊 Customer Segmentation Dashboard (Power BI)

#📌 Overview
This project presents an interactive Customer Segmentation Dashboard built using Power BI. 
The dashboard helps analyze customer data to uncover insights related to revenue, demographics, purchasing behavior, and geographic distribution.
It is designed to support data-driven decision-making by identifying high-value customers, understanding sales trends, and exploring customer segments effectively.

#🎯 Objectives
Identify top customers contributing to revenue
Analyze customer distribution by country/region
Understand purchase behavior patterns
Evaluate the impact of demographic factors (age, gender, marital status)
Track quarterly sales and budget performance

#📈 Key Features
🔍 Dynamic Filters (Slicers)
Filter data by Country, Age Group, Gender, and Quarter
👤 Top Customers Analysis
View top customers by revenue and highest tax contributors
🌍 Geographical Insights
Country-wise customer distribution visualization
💰 Revenue Analysis
Identify top-performing customers and revenue trends
🧑‍🤝‍🧑 Customer Demographics
Analyze customers by marital status, age group, and gender
🛒 Purchase Behavior
Understand order frequency (1 order, multiple orders, etc.)
📊 Quarterly Sales vs Budget
Compare sales performance across quarters
🛠️ Tools & Technologies Used
Power BI – Data visualization and dashboard creation
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Measures and calculations
Excel / CSV Dataset – Data source

#🚀 How to Use
Download the .pbix file from this repository
Open it using Power BI Desktop
Use slicers to interact with the dashboard
Explore insights across different visualizations

#📌 Insights Generated
Top customers contribute a significant portion of total revenue
Majority of customers belong to specific regions (e.g., US, Australia)
Customers with fewer orders still generate high revenue in some cases
Demographic factors influence purchasing behavior and revenue trends
